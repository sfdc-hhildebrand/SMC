Provides the classes modeling a finite state machine. In SMC, a finite state machine consists of

  1. {@link net.sf.smc.model.SmcMap Map}s which consist of
  2. {@link net.sf.smc.model.SmcState State}s which consist of
  3. {@link net.sf.smc.model.SmcTransition Transition}s which consist of
  4. {@link net.sf.smc.model.SmcParameter Parameter}s and {@link net.sf.smc.model.SmcGuard Guard}s which consist of
  5. {@link net.sf.smc.model.SmcAction Action}s.

The {@link net.sf.smc.model.SmcVisitor} abstract class implements the GoF Visitor pattern (p. 331). This pattern is used by the generator package to generate target language code from the FSM model.

See the programmers manual for SMC's derivation from the Gof State pattern.
